# 🎬 CineSearch - Movie & TV Show Search

A modern movie and TV show search application with watchlist functionality and video playback.

## 🚀 Features

- **Movie/TV Search**: Search movies and TV shows using OMDb API
- **Detailed Information**: Comprehensive movie details including ratings, cast, plot
- **Video Playback**: Integrated YouTube trailer playback with fallback system
- **Watchlist Management**: Add/remove movies from personal watchlist
- **Fullscreen Mode**: Immersive video viewing experience
- **Animated Space Background**: Beautiful canvas-based starfield animation
- **Responsive Design**: Optimized for all devices and screen sizes
- **Dark Theme**: Modern dark UI for cinematic experience

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **API**: OMDb API for movie data
- **Video**: YouTube iframe API for trailers
- **Graphics**: HTML5 Canvas for space animations
- **Storage**: localStorage for watchlist persistence
- **Icons**: Font Awesome

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start searching for your favorite movies and shows
4. Add movies to your watchlist for later viewing

## 🔧 API Configuration

This app uses the **OMDb API**. The API key is already configured in the application.

**Note**: The OMDb API has rate limits. For production use, consider getting your own API key.
